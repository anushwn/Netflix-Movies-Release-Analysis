This project analyzes movie data using Python to uncover insights about ratings, popularity, and genre trends. The analysis includes data cleaning, feature engineering, and exploratory data analysis to understand how different factors influence movie performance.

Key metrics such as average rating, popularity distribution, genre frequency, and release trends were examined to identify patterns in the dataset. Visualizations highlight relationships between vote average, popularity, and movie genres, helping to reveal which categories tend to perform better.

The project demonstrates practical skills in Pandas, data preprocessing, data visualization, and analytical thinking, showcasing how raw datasets can be transformed into meaningful insights relevant to real-world data analyst workflows.
